# @four-bytes/four-opencode-notification

Multi-provider webhook notification plugin (Discord, Slack, MS Teams, generic) — custom notify_send tool + event hooks + team patterns. Source: Perplexity P50.

## Status

Sprint P14 (bootstrap), in Planung.

## Installation

```bash
bun install @four-bytes/four-opencode-notification
```

## Konfiguration

```typescript
// opencode.config.ts
export default {
  plugins: [
    require("@four-bytes/four-opencode-notification"),
  ],
};
```
